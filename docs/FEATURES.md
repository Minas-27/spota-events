## üé´ Attendee Features

### 1. Event Exploration
- **Real-time Discovery:** A dynamic home screen showing live events from Firestore.
- **Search & Filter:** Intelligent search bar to find events by title, and category chips to filter by interest (Music, Tech, Cultural, etc.).
- **Detailed View:** Comprehensive event pages with high-quality images, descriptions, location, and pricing.

### 2. Booking Workflow
- **Ticket Selection:** Choose quantity and view availability in real-time.
- **Secure Payment:** Integration with **Chapa**, allowing users to pay using local Ethiopian methods.
- **Instant Confirmation:** Automated SMS confirmations sent via **AfroMessage** upon successful payment.
- **Digital Tickets:** A dedicated "My Tickets" section to store and view all active and past bookings.

### 3. Personalization
- **Profile Management:** Update personal details, bios, and profile pictures.
- **Notifications:** In-app notification center for booking updates and platform news.
